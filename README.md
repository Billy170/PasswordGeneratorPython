User Interface Overview

Password: Displays the generated password

Encrypted: Shows the encrypted version of the password

Length: Controls password length

Power: Password strength selector (Low / Medium / Strong)

Generate: Creates a new password

Encrypt: Encrypts the password using the key

Decrypt: Decrypts encrypted text using the key

Key: Numeric key used for encryption/decryption

Save to file: Saves output to a local file

How to Use

Select the desired password length

Choose the strength level

Click Generate to create a password

Enter a key

Click Encrypt to encrypt the password

Use Decrypt with the same key to retrieve the original password

Use Copy buttons to copy values to the clipboard

(Optional) Enable Save to file to store results
